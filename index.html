Sistema completo de votación

    Voting es un proyecto realizado como práctica de entrega para la asignatura Arquitectura del Software de la Universidad de Oviedo.

    El sistema es capaz de completar el proceso para cualquier tipo de votación. Desde leer datos de censos, pasando porque el votante consulte su colegio electoral, vote y consulte los resultados
